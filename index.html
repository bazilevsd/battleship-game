How to play the game.

There are two players. Each player positions its ships secretly from the opponent. After the ships have been positioned, the game proceeds in a series of rounds. turn changes when player misses the ship. In each round, each player takes a turn to click on a target square in the opponent's grid which is to be shot at. If it is a "hit", the square turns red. If all of a player's ships have been sunk, the game is over and their opponent wins. If all ships of both players are sunk by the end of the round, the game is a draw.
